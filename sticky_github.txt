GitHub Repository Link:


Example: https://github.com/ceaganvs/sticky-notes-app

Instructions:
1. Create a new repository on GitHub named "sticky-notes-app"
2. Make sure it's set to PUBLIC
3. In your local project folder, run the following commands:

   git init
   git add .
   git commit -m "Initial commit: Django sticky notes application"
   git branch -M main
   git remote add origin https://github.com/ceaganvs/sticky-notes-app.git
   git push -u origin main

4. After pushing, paste your repository URL at the top of this file

IMPORTANT NOTES:
- DO NOT run "collectstatic" before pushing to GitHub
- The staticfiles/ folder is already excluded via .gitignore
- Anyone cloning the repo will run "collectstatic" on their own machine
- The .gitignore also excludes: venv/, db.sqlite3, __pycache__/
